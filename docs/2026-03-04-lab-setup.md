# 2026-03-04 - Lab Setup

**Objective:**  

Establish initial physical and logical connectivity across all devices and create a structured repository for automation development.

## Equipment & Software

- Adtran NetVanta 5660
- Juniper SRX240
- Cisco Catalyst 3650
- 2 Ubuntu 22.04 servers
- TFTP64 on Windows laptop with WSL2 running VS Code
- Git & GitHub for version control

## Physical & Network Setup

- Catalyst 3650 connected to Adran 5660 Gi2/0/48 (192.168.254.2/30) ↔ Gi0/3 (192.168.254.1/30)
- Adtran 5660 connected to Juniper SRX240 Gi0/2 (10.0.0.2/30) ↔ ge-0/0/1 (10.0.0.1/30)
- Ubuntu servers connected to Catalyst VLANs:
  - Server1: VLAN10, 192.168.10.2/24
  - Server2: VLAN10, 192.168.10.3/24
- Windows host connected to Catalyst VLANS:
  - Host1: VLAN20, DHCP
